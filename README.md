# credit_default_prediction
# Предсказание дефолта по кредиту (Credit Default Prediction)

**Задача**: бинарная классификация — предсказать, будет ли просрочка по кредиту (класс 1 — дефолт)

**Метрика**: F1-score по классу 1

**Достигнутые результаты**:
- Лучшая модель: CatBoost + оптимизация порога классификации
- F1 на валидации (класс 1): **0.4804** (порог 0.272)
- После ручной настройки порога: **F1 до 0.50–0.53** (порог ~0.34–0.36)
- Самописная логистическая регрессия: F1 ≈ 0.1994
- CatBoost значительно лучше линейной модели благодаря учёту взаимодействий признаков

**Основные шаги**:
- Обработка аномалий (99999999 → NaN, Credit Score /10)
- Генерация признаков: отношения долга к доходу, комбинации низкого скора + высокой нагрузки
- KNNImputer + StandardScaler
- CatBoost с class_weight='Balanced' + early stopping
- Подбор порога для максимизации F1
